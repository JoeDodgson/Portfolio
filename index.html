My portfolio

Bookworm

A webpage for book lovers to find information about books and add them to their read list. Users select a target read date for each book they add to their read list.

Technologies adopted:

  • AJAX calls to retrieve API data
  • CSS Grid Layout framework
  • jQuery JavaScript library

Weather dashboard

The user searches for any city to see the current weather and a 5 day weather forecast. The page stores and displays the user's searched cities.

Technologies adopted:

  • AJAX calls to retrieve API data
  • CSS Grid Layout framework
  • jQuery JavaScript library

Coding quiz

A multiple choice coding quiz, including a page which displays the Highscores. The user submits their score which is added to other scores in the local storage.

Technologies adopted:

  • AJAX calls to retrieve API data
  • CSS Grid Layout framework
  • jQuery JavaScript library

Password generator

Generates a random password for a user based character type selection criteria. The user selects character types to include in the password through a series of prompts.

Technologies adopted:

  • Event listeners to trigger functions
  • Use of alerts, confirms and prompts
