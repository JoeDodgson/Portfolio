My portfolio

Fitness buddy

A website for gym-goers to find information about exercise technique. Users login or sign up, search for exercises and can add exercises to their favourites list.

Technologies adopted:

  • Express server
  • Heroku deployment
  • SQL database and Sequelize
  • Pug template engine

Bookworm

A webpage for book lovers to find information about books and add them to their read list. Users select a target read date for each book they add to their read list.

Technologies adopted:

  • AJAX calls to retrieve API data
  • CSS Grid Layout framework
  • jQuery JavaScript library

  • Use of setInterval and setTimeout JavaScript methods
